👋 About this resource

This is a chapter-by-chapter teaching package for Nur al-Idah — the classical introductory Hanafi text by Imam Hasan Shurunbulali. The package is designed for an adult, mostly Urdu-speaking cohort learning practical fiqh in English + Roman Urdu.

For each chapter you get four deliverables:

  • 📝 Deep Notes — Ramsha's personal learning material. Read this first. Part A explains the chapter at depth; Part B is the teaching toolkit.
  • 🎓 Teacher Notes — master overview chart, concept-by-concept Mermaid flowcharts, definition cards (English + Roman Urdu + Arabic), common mistakes, Qur'an & Hadith citations.
  • 🧑‍🎓 Student Handout — printable, with exercises and homework.
  • 📽️ Slide Deck — reveal.js slides for classroom projection. Use arrow keys; press F for fullscreen.

Source constraint: every ruling, verse, and hadith comes only from Nur al-Idah itself — no outside sources are imported.

📚 Book I — Purification (كِتَابُ الطَّهَارَة)

Book I of Nur al-Idah covers the foundational rulings on ritual purity — water, cleaning oneself, wudu, ghusl, tayammum, wiping over leather socks, menstruation/postnatal bleeding, and filth (najasah). It is broken into 8 sub-chapters here:

📖 How to use these materials

Each chapter has four files. Use them in this order:

  1. 📝 Deep Notes — start here to understand the chapter at depth (Part A) and prepare to teach it (Part B).
  2. 🎓 Teacher Notes — chart-driven reference for the in-class flow.
  3. 🧑‍🎓 Student Handout — print this for classmates; includes exercises and homework.
  4. 📽️ Slide Deck — project this while teaching.
