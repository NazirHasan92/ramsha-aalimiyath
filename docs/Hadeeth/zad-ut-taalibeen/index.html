Zad ut-Taalibeen — Provisions for the Seekers

Teaching materials by eMahad · A manual of Prophetic Aḥādīth with commentary · For adult learners (English + Roman Urdu)

👋 About this resource

This is a teaching package for Zad ut-Taalibeen by Shaykh Muḥammad ʿĀshiq Ilāhī al-Bulandshehrī (raḥimahullāh), drawing on the English translation and commentary by Mufti Abdur-Rahman ibn Yusuf (Provisions for the Seekers, White Thread Press).

We cover the short aḥādīth of Part 1 — the pithy one-line sayings of the Prophet ﷺ that scholars memorise as foundational wisdom (target: the first 100, ten per chapter). Chapters 1–6 (Aḥādīth 1–60) are complete.

For each chapter you get four deliverables:

  • 📝 Deep Notes — Ramsha's personal study material. Part A = understand the chapter; Part B = how to teach it.
  • 🎓 Teacher Notes — chart-driven in-class reference: master overview, per-ḥadīth charts, definition cards, common confusions, Qurʾān connections.
  • 🧑‍🎓 Student Handout — printable, exercise-focused, for classmates.
  • 📽️ Slide Deck — reveal.js slides for classroom projection.

📖 How to use these materials

Each chapter has four files. Use them in this order:

  1. 📝 Deep Notes — start here to understand the chapter at depth (Part A) and prepare to teach it (Part B).
  2. 🎓 Teacher Notes — chart-driven reference for the in-class flow.
  3. 🧑‍🎓 Student Handout — print this for classmates; includes reflection prompts and exercises.
  4. 📽️ Slide Deck — project this while teaching.
