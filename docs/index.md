# Ramsha's eMahad Notes

Page-by-page learning materials for the **8-year Aalim course** at eMahad. Each chapter ships **four deliverables**: deep notes (the personal learning material in Markdown), teacher view (HTML), student handout (HTML printable), and slides (HTML presentation).

---

## Subjects

| Subject | Book | Author / Tradition | Progress |
|---|---|---|---|
| **Arabic** | Madinah Arabic Book 1 | Dr. V. Abdur Rahim | Ch 1-5 complete (of 15 target) |
| **Fiqh** | Nur al-Idah · Book I (Purification) | Hasan Shurunbulali (Hanafi) | Ch 1-2 complete + Ch 3a in progress (of 8 target) |
| **Hadeeth** | Zad ut-Taalibeen (Part 1) | — | Ch 1-4 complete = 40 aḥādīth (of 10 ch / 100 hadiths target) |

---

## Deliverable types per chapter

| Type | Purpose | Format |
|---|---|---|
| **Deep notes** | Ramsha's personal learning material — Part A (understand) + Part B (teach back) | Markdown (this site renders them) |
| **Teacher view** | Structured chapter for teaching the cohort | HTML with custom CSS |
| **Student handout** | Printable / shareable summary for students | HTML with print-friendly CSS |
| **Slides** | Presentation slides for classroom delivery | HTML slideshow |

Within each chapter's deep-notes page, links at the top jump to the matching teacher / student / slides view.
